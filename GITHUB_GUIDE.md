# 🔧 뚝딱공작소 - GitHub에서 EXE 자동 생성하기

## ⚡ 정말 쉬운 3단계! (클릭만!)

---

## 1️⃣ GitHub 계정 만들기 (2분)

### Step 1: GitHub 접속
```
https://github.com
```

### Step 2: Sign up 클릭
우측 상단의 **"Sign up"** 클릭

### Step 3: 이메일 입력
- 이메일 주소 입력
- 비밀번호 입력
- "Create account" 클릭

### Step 4: 이메일 인증
받은 메일에서 인증 링크 클릭

✅ **GitHub 계정 완성!**

---

## 2️⃣ Repository 생성 (1분)

### Step 1: GitHub 로그인
https://github.com 로그인

### Step 2: 새 Repository 만들기
- 우측 상단 **"+"** 클릭
- **"New repository"** 클릭

### Step 3: 정보 입력
- **Repository name**: `ttaktak-studio` 입력
- **Public** 선택 (자동 선택됨)
- **"Create repository"** 클릭

✅ **Repository 생성 완료!**

---

## 3️⃣ 파일 업로드 (2분)

### Step 1: 파일 선택
당신의 컴퓨터에서:
```
ttaktak-studio 폴더의 모든 파일:
- app.py
- requirements.txt
- run.bat
- README.md
- templates/ (폴더)
- uploads/ (폴더)
- .github/ (폴더 - 이미 포함됨)
```

### Step 2: GitHub에 업로드
1. GitHub에서 만든 repository 페이지 접속
2. **"Add file"** → **"Upload files"** 클릭
3. 파일들을 **드래그 & 드롭** (또는 클릭해서 선택)
4. 아래 **"Commit changes"** 클릭

✅ **파일 업로드 완료!**

---

## 4️⃣ 자동 빌드 시작 (기다리기)

### 자동으로 실행됨!

파일을 업로드하면 GitHub이 자동으로:
1. **Actions** 탭에서 빌드 시작
2. Python 설치
3. 라이브러리 설치
4. EXE 생성

### 상태 확인
1. Repository의 **"Actions"** 탭 클릭
2. **"Build EXE"** 작업 확인
3. 상태:
   - 🟡 **진행 중** (기다림)
   - 🟢 **완료** (다음 단계!)
   - 🔴 **실패** (다시 시도)

⏱️ **소요 시간**: 3-5분

---

## 5️⃣ EXE 다운로드! 🎉

### Step 1: 빌드 완료 확인
Actions 탭에서 녹색 체크 표시 🟢 확인

### Step 2: Artifact 다운로드
1. "Build EXE" 클릭
2. 아래로 스크롤
3. **"뚝딱공작소-EXE"** 섹션 찾기
4. **다운로드** 버튼 클릭

### Step 3: 압축 풀기
1. 다운로드된 ZIP 파일 열기
2. **"뚝딱공작소.exe"** 추출

✅ **완료!** 이제 이 EXE 파일을 배포하면 됩니다!
