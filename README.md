# 🔧 뚝딱공작소 - AI 이미지 업스케일러

**고급 AI 알고리즘을 사용한 이미지 고화질 업스케일링 도구**

---

## ⚡ 빠른 시작 (2단계)

### 1️⃣ **EXE 파일 만들기** (처음 한 번만)

`build-simple.bat` **더블클릭** 👈

```
자동으로:
✅ Python 확인
✅ 라이브러리 설치 (5-10분)
✅ EXE 생성 (1-3분)
✅ dist/뚝딱공작소.exe 완성!
```

### 2️⃣ **EXE 실행하기**

`dist` 폴더에서 `뚝딱공작소.exe` **더블클릭**

```
자동으로:
✅ 명령 프롬프트 열림
✅ 웹 서버 시작
✅ 브라우저에서 http://localhost:5000 열림
✅ 앱 사용 시작!
```

---

## 📋 시스템 요구사항

- **Windows 7 이상**
- **Python 3.8 이상** (EXE 만들 때만 필요)
- **최소 4GB RAM**

---

## 🎯 사용법

### 이미지 처리

1. **이미지 선택**
   - 드래그 & 드롭 또는 클릭

2. **모드 선택**
   - 고화질 복원: 크기 유지, 품질만 개선
   - 2배 확대: 2배 크기로 확대
   - 4배 확대: 4배 크기로 확대

3. **처리 시작**
   - "🚀 처리 시작" 버튼 클릭

4. **다운로드**
   - "💾 다운로드" 클릭

## 🚀 EXE 파일 배포

완성된 `뚝딱공작소.exe` 파일은:
- 📧 이메일로 배포 가능
- ☁️ 클라우드에 업로드 가능
- 💾 USB에 저장 가능
- 🌐 웹사이트에 게시 가능

**받은 사람이 더블클릭하면 바로 실행!**

Python 설치 필요 없음! ✅

---

## ⚙️ 문제해결

### Python이 설치되지 않음

1. https://www.python.org/downloads/ 다운로드
2. 설치 시 **"Add Python to PATH"** 체크!
3. 컴퓨터 재시작

### EXE 빌드 실패

- 관리자 권한으로 `build-simple.bat` 실행
- 또는 명령 프롬프트를 **관리자 권한**으로 열어서 실행

### 앱 실행 오류

- `뚝딱공작소.exe` 더블클릭
- 명령 프롬프트 창 닫지 말기
- 브라우저에서 http://localhost:5000 접속
